

# 팝업 위젯은 기본적으로 라인트레이스를 발사해 라인트레이스와 충돌되는 오브젝트에 한해서만
  팝업창을 띄움(ex. 무기, 오브젝트, NPC대화 등)
  오브젝트나 NPC의 메시 충돌에 Visible이 무시 또는 겹침으로 설정되어 있으면 팝업 위젯이 활성화되지 않음.

# 인벤토리와 퀵슬롯의 슬롯수를 조정하고 싶을 때, 캐릭터 헤더파일에 있는 인벤토리와 퀵슬롯의 맥스사이즈를 조정하고,
  에디터 ActionSlotBar와 InventoryBar에서 슬롯의 인덱스를 조정해야 함.
  캐릭터가 인벤토리와 퀵슬롯을 가지도록 코드를 짰으나, 일반적으로 컨트롤러가 인벤토리와 퀵슬롯을 가지는 것이 좋음.
